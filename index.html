eGov-Data 프로젝트는?

Spring Data 프로젝트의 주요 목표는 스프링 기반 애플리케이션이 데이터 접근 기술을 보다 손쉽게 사용할 수 있게 해주는 것입니다. eGov-Data 프로젝트는 표준프레임워크에서 사용하는 Hibernate와 iBatis 기반 데이터 접근 계층에 Spring Data의 핵심 기능을 제공합니다.

이 프로젝트는 두개의 세부 모듈로 구성되어 있습니다.

  • eGov Data iBatis
  • eGov Data Hibernate

eGov Data iBatis

public class BookRepository extends SqlMapRepository<Book, Integer> {

  Book getById(Integer id);
  
  List getAll();

}

아이바티스를 사용하신다면 위와 같이 SqlMapRepository를 상속받는 인터페이스를 만들고 그 안에 SQL 키값과 동일한 이름을 가진 메서드를 정의하시면 됩니다. 구현체는 만드실 필요 없습니다. 

<?xml version="1.0" encoding="UTF-8"?>
<beans xmlns="http://www.springframework.org/schema/beans"
	xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
	xmlns:ibatis="http://www.egovframe.org/schema/data/ibatis"
	xsi:schemaLocation="http://www.springframework.org/schema/beans http://www.springframework.org/schema/beans/spring-beans.xsd
		http://www.egovframe.org/schema/data/ibatis http://www.egovframe.org/schema/data/ibatis/egov-ibatis.xsd">
    
    <ibatis:repositories base-package="egov.data.ibatis.sample.repository"/>
    
</beans>

이렇게 간단한 XML 빈 설정만 추가하시면 구현체는 자동으로 생성됩니다.

@Service
public class DefaultBookService implements BookService {

  @Autowired BookRepository repository;

}

이렇게 BookRepository 인터페이스 타입의 빈이 등록되어있는 것처럼 어디서든지 @Autowired/@Inject/@Resource 등을 사용하여 주입받아 사용하시면 됩니다.

eGov Data iBatis의 자세한 내용이 궁금하시면 가이드 문서를 참고하시기 바랍니다.

eGov Data Hibernate

public class BookRepository extends HibernateRepository<Book, Integer> {


}

eGov Data Hibernate를 사용하시면 위와 같이 HibernateRepository를 상속받은 인터페이스만 만들면 기본적인 CRUD 기능과 Paging 및 Sorting 기능을 제공합니다.

<?xml version="1.0" encoding="UTF-8"?>
<beans xmlns="http://www.springframework.org/schema/beans"
       xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
       xmlns:hibernate="http://www.egovframe.org/schema/data/hibernate"
       xsi:schemaLocation="http://www.springframework.org/schema/beans http://www.springframework.org/schema/beans/spring-beans.xsd
            http://www.egovframe.org/schema/data/hibernate http://www.egovframe.org/schema/data/hibernate/egov-hibernate.xsd">

    <hibernate:repositories base-package="whiteship" />

</beans>

이렇게 간단한 XML 빈 설정만 추가하시면 구현체는 자동으로 생성됩니다.

@Service
public class DefaultBookService implements BookService {

  @Autowired BookRepository repository;

}

이렇게 BookRepository 인터페이스 타입의 빈이 등록되어있는 것처럼 어디서든지 @Autowired/@Inject/@Resource 등을 사용하여 주입받아 사용하시면 됩니다.

eGov Data Hibernate의 자세한 내용이 궁금하시면 가이드 문서를 참고하시기 바랍니다.

주요 기능

  • 기본 CRUD 메서드 구현 제공.
  • 쿼리 메서드 이름을 사용한 동적 쿼리 생성.
  • 커스텀 리포지토리 코드 연동.
  • 커스텀 네임스페이스 제공.
