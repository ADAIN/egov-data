eGov-Data-iBatis Quick Start

eGov-Data-iBatis 설치:

  1. 프로젝트 다운로드

    git clone https://github.com/spring-sprout/egov-data.git

  2. Maven을 사용해 jar 생성

    mvn jar:jar

  3. classpath에 target/egov-data-ibatis.[version].jar 추가
